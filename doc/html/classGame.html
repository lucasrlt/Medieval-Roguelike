Detailed Description

Classe s'occupant de créer le jeu. C'est cette classe qui gère l'intéraction entre les modules. Une instance du jeu contient une Tilemap avec la salle courante, ainsi que les différentes entités (ennemis et joueur). De plus, le jeu stocke tous les projectiles envoyés, et les actualise.
