Detailed Description

Classe gérant la TileMap. Une tilemap est un tableau de tiles qui provient d'un fichier .tmx créé avec le logiciel Tiled. Les tiles sont des images carrées que l'on assemble pour créer des niveaux. Ce module gére la récupération et le traitement de TileMaps depuis un fichier .tmx pour gérer les niveaux dans le jeu.
